生成二维码

接口地址
/app/dyBillQrcode/generateQRCode/{billNo}/{billAddressId}
请求方式

POST

接口描述

二维码扫码信息

{
"billNo":"20171113061245", //运单ID 
"billUnitId":"13216555", // 收发货方id，
"billAddressId": "12312324", // 装卸货地址id,
"addressType":"1" //1 :发货,2: 收货
}
请求参数

格式：JSON

参数名 类型 描述 是否必填
billNo Long 运单号 TRUE
billAddressId Long 运单地址主键ID TRUE
响应参数

格式：JSON

参数名 类型 描述
String Base64
