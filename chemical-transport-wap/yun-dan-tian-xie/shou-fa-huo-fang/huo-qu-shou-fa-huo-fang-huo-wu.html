获取收发货方货物

#

获取收发货方货物

接口地址
/app/dyBillShipReceiveUnit/getDyShipReceiveUnitAndDyBillChemical/{billNo}
请求方式

POST

接口描述
请求参数

格式：JSON

参数名 类型 描述 是否必填
billNo String 运单号 Y
unitType Integer 1:发货方；2:收货方 N
响应参数

格式：JSON

参数名 类型 描述
status Integer 状态码 1:成功；0：失败
message String 响应消息
调用示例
/app/dyBillShipReceiveUnit/getDyShipReceiveUnitAndDyBillChemical/123456

参数

{unitType：1}
